# IZOQ Systems LTD

Africa's crypto payments ecosystem — built for the 2026 Nigeria market.

## Projects

| Folder | Description | Deploy |
|--------|-------------|--------|
| `/izoq` | Main website & landing page | Root Directory: `izoq` |
| `/izoq-inlet` | Merchant payment gateway | Root Directory: `izoq-inlet` |
| `/izoq-outlet` | User wallet app | Root Directory: `izoq-outlet` |

## Deploy on Vercel
Each folder is an independent Vite + React project.
Import the repo 3 times on Vercel, setting the Root Directory to each folder.

Install Command: `npm install`
Build Command: `npm run build`
Output Directory: `dist`
