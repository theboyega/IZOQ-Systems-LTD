# IZOQ Outlet — User Wallet

The user-facing wallet application for the IZOQ payments ecosystem.

## Pages
- **My Wallet** — Multi-chain portfolio, send/receive, withdraw to bank
- **Activity** — Personal transaction history
- **Verification** — Sumsub KYC (unlocks all wallet features)

## Tech Stack
- React 18 + Vite
- Pure CSS-in-JS

## Getting Started

```bash
npm install
cp .env.example .env
npm run dev
```

## Deploy to Vercel
1. Push repo to GitHub
2. Vercel → New Project → Import repo
3. Set **Root Directory** to `izoq-outlet`
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Deploy
